# **App Name**: Evolution CTF

## Core Features:

- Start Time Recording: Records the timestamp when the user starts the CTF.
- Challenges Listing: Displays a list of challenges with descriptions and difficulty levels.
- Progressive Hints: Provides hints for each challenge, revealed progressively.
- Flag Submission: Accepts case-sensitive flag submissions and validates them.
- Solve Recording: Records successful challenge solves with timestamps.
- Admin Interface: Password-protected admin interface for seeding flags and viewing results.
- Scoreboard Ranking: Ranks players based on score and time.

## Style Guidelines:

- Primary color: A deep, desaturated teal (#468499) to represent the past, fading into the future.
- Background color: A light, off-white (#F0F8FF), providing a clean, modern look.
- Accent color: A brighter, complementary orange (#E07A5F) to highlight interactive elements.
- Headline font: 'Space Grotesk', a sans-serif for a techy feel, suitable for titles. Body Font: 'Inter', a sans-serif to maintain the tech feel throughout all text.
- Use icons to represent the difficulty levels of each challenge (Past, Present, Future).
- A clean, intuitive layout with clear navigation between challenges and the scoreboard.
- Subtle transitions and animations to indicate progress and feedback on flag submissions.
