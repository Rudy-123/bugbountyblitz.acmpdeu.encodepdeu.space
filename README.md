# CTF ACM x Encode

## Features



   
- **Team-based Competition**: Each team gets 1 hour to solve challenges
- **Live Leaderboard**: Real-time rankings synced with Google Sheets
- **Scoring System**: Points for solving challenges, penalties for hints
- **Timer**: 1-hour countdown with time-up notifications

## Routing Changes

The admin and answers sections are no longer accessible through navigation buttons on the site. They can only be accessed by directly navigating to `/admin` and `/answers` routes respectively.
